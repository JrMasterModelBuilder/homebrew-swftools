# Usage

## Tap

```sh
brew tap JrMasterModelBuilder/swftools
```

## Install Stable

```sh
brew install swftools-stable
```

## Install Dev

```sh
brew install swftools-dev
```

## Install Head

```sh
brew install --HEAD swftools-head
```
